ยินดีต้อนรับ — pedal

หน้านี้เป็นเพียงหน้าแนะนำบน GitHub Pages (เว็บไฟล์คงที่) ไม่ใช่ตัวแอป Next.js ที่รันในโปรเจกต์นี้

โปรเจกต์นี้ใช้ Next.js + Prisma + ฐานข้อมูล ต้องรันบนเซิร์ฟเวอร์ Node.js จึงไม่สามารถ deploy แบบเว็บ HTML ธรรมดาบน GitHub Pages ได้โดยตรง

ดูซอร์สโค้ด: github.com/Peemyeet/pedal

แนะนำ: ใช้ Vercel (เชื่อมกับ repo นี้) โดยตั้งค่า DATABASE_URL เป็นฐานข้อมูลที่โฮสต์ได้ (เช่น Neon, Vercel Postgres) แล้วรัน npx prisma migrate deploy / prisma db push ตามที่ใช้
